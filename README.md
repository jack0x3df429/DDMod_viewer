# DDMod_viewer
遊戲Darkest Dungeon的模組管理器

因原版模組排序太過難用，故自製一個管理器:
目前能做到:
1.解析遊戲加密文件
2.網頁瀏覽遊戲文件
3.模組依規則自行排序
4.重新寫回檔案
